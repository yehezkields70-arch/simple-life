# Simple Life — Health Reminder App

Simple Life adalah aplikasi web pengingat kesehatan harian yang dibuat sebagai Final Project mata kuliah Pemrograman Web, Program Studi Informatika, Universitas Dipa Makassar.

---

## Tentang Aplikasi

Aplikasi ini dirancang untuk membantu pengguna menjaga kebiasaan hidup sehat melalui pengingat yang bisa diatur sesuai kebutuhan, seperti minum air putih, olahraga, waktu tidur, dan lainnya. Semua data tersimpan langsung di browser pengguna tanpa memerlukan server atau database eksternal.

---

## Fitur

- Tambah, edit, dan hapus reminder
- Alarm dengan notifikasi browser tepat waktu
- Filter reminder berdasarkan kategori
- Statistik ringkas: total reminder, alarm aktif, dan reminder yang berlaku hari ini
- Data tersimpan otomatis di browser (tidak hilang saat browser ditutup)

## Teknologi yang Digunakan

- HTML5, CSS3, JavaScript (Vanilla)
- localStorage — untuk menyimpan data reminder di browser
- Web Notifications API — untuk menampilkan notifikasi alarm

Tidak menggunakan framework atau backend. Aplikasi berjalan sepenuhnya di sisi klien.
